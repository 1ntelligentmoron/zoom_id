Brief Introduction:
Electron app with Python 3 as backend for customisable timetable with easy Zoom PMI copying.

Requirements and Important Info:
A. Google Chrome is needed for the app to function.
B. This app is may not work as intended if the OS of your PC is not Windows 10.
C. Resizing the window should solve all overflowing issues.
